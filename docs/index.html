1、form 下面只有一个 input 时回车键刷新页面

原因是触发了表单默认的提交行为，给el-form 加上@submit.native.prevent就行了。

2、表格固定列最后一行显示不全

image.png

这种情况有时在宽度刚好处于临界值状态时会出现。因为固定列是独立于表格body动态计算高度的，出现了固定列高度小于表格高度所以造成最后一行被遮挡。
